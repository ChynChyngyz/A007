# AnyDesk-like Remote Desktop Application

This project is a cross-platform remote desktop application inspired by **AnyDesk**, built using **Flutter** for the client side and **Django** for the backend.

The goal of the project is to provide fast, secure, and reliable remote access to another device over the internet.

---

## Tech Stack

### Frontend (Client)
- **Flutter**
- Dart
- WebSocket / WebRTC (for real-time communication)
- Cross-platform support (Windows, macOS, Linux, Android, iOS)

### Backend (Server)
- **Django**
- Django REST Framework
- WebSockets (Django Channels)
- PostgreSQL / SQLite
- JWT authentication
